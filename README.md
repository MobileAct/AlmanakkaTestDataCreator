# AlmanakkaTestDataCreator
Almanakkaの日付生成コードのテストをするためのテストデータ作成機

## Test Data
Test Data is generated in `/docs`directory.

### FileName rule
Date format: `{Year}-{Month}-{Day}`  
FileName format: `{StartDate}_{EndDate}_{DayOfWeekOrderStart}_{IsContainDaysOfDifferentMonth}.json`
